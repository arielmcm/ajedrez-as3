# ajedrez-as3 

Juego de ajedrez escrito en ActionScript 3.

Última modificación: Marzo de 2012

El juego no ha sido terminado aún, valida todos los movimientos legales (peón al paso, enroques, piezas clavadas, promociones de peón etc), jaques y mates (no se muestra en interfaz gráfica, sólo por consola).

Falta la validación de los turnos de los jugadores.

Eres libre de usar y modificar el código a tu gusto.
